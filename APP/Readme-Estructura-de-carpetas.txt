Descripcion y estructura de Carpetas
1. assets: En esta carpeta estan todos los archivos (css, javascript, sass y webfonts) para darle diseño, funcionalidad y nuevos elementos a las paginas web
    1.1 assets>css: En esta carpeta estan todos los archivos (css) para el "Diseño" de todas las paginas web que se vayan creando
        1.1.1 assets>css>fontawesome-all.min.css: Este archivo es una libreria para darle estilo a las fuentes 
        1.1.2 assets>css>header.css: Este archivo sirve para darle estilo al header
        1.1.3 assets>css>main.css: Este archivos sirve para darle estilo al contenido principal
        1.1.4 assets>css>noscript.css: Este archivo sirve para anular todos los estilos que tiene por defecto el navegador

    1.2 assets>js: En esta carpeta estan todos los archivos (js) para darle "funcionalidad y animaciones" a todas las paginas web que se vayan creando
        1.2.1 assets>js>header.js: Este archivo sirve para darle funcionalidad y animacion el header
        1.2.2 assets>js>jquery.min.js: Este archivo es la libreria principal de jquery
        1.2.3 assets>js>main.js: Este archivo sirve para darle funcionalidad y animacion a todos los elementos del contenido principal
        1.2.4 assets>js>"Resto de archivos .js": Son librerias utilies que ayudan a darle animaciones a la pagina web

    1.3 assets>webfonts: En esta carpetan estan todos los archivos para usar tipos de fuentes en el estilo y diseño de las paginas webs

2. Facturacion: En esta carpetan estaran todos los formularios del modulo de Facturacion
    -Se identifica con el prefijo: FA_

3. GestionUsuario: En esta carpetan estaran todos los formularios del modulo Gestion de Usuarios
    -Se identifica con el prefijo: GU_

4. images: En esta carpeta estaran todas las imagenes necesarias del Aplicativo
    4.1 images>Facturacion: En esta carpeta estaran todas las imagenes especiales para el modulo Facturacion
    4.2 images>GestionUsuario: En esta carpeta estaran todas las imagenes especiales para el modulo GestionUsuario
    4.3 images>OrdenTrabajo: En esta carpeta estaran todas las imagenes especiales para el modulo OrdenTrabajo
    4.4 images>RRHH: En esta carpeta estaran todas las imagenes especiales para el modulo RRHH
    4.5 images>Vehiculos: En esta carpeta estaran todas las imagenes especiales para el modulo Vehiculos

5. Libreria: En esta carpeta estaran todos los archivos para darle alguna funcionalidad o estilos extra al Aplicativo
    5.1 Libreria>css: En esta carpeta estaran todos los archivos css especciales para cada modulo
        5.1.1 Libreria>css>Facturacion: En esta carpeta estaran todos los archivos css especiales para el modulo Facturacion
        5.1.2 Libreria>css>GestionUsuario: En esta carpeta estaran todos los archivos css especiales para el modulo GestionUsuario
        5.1.3 Libreria>css>OrdenTrabajo: En esta carpeta estaran todos los archivos css especiales para el modulo OrdenTrabajo
        5.1.4 Libreria>css>RRHH: En esta carpeta estaran todos los archivos css especiales para el modulo RRHH
        5.1.5 Libreria>css>Vehiculos: En esta carpeta estaran todos los archivos css especiales para el modulo Vehiculos
    5.2 Libreria>js: En esta carpeta estaran todos los archivos js especiales para cada modulo
        5.2.1 Libreria>js>Facturacion: En esta carpeta estaran todos los archivos js especiales para el modulo Facturacion
        5.2.2 Libreria>js>GestionUsuario: En esta carpeta estaran todos los archivos js especiales para el modulo GestionUsuario
        5.2.3 Libreria>js>OrdenTrabajo: En esta carpeta estaran todos los archivos js especiales para el modulo OrdenTrabajo
        5.2.4 Libreria>js>RRHH: En esta carpeta estaran todos los archivos js especiales para el modulo RRHH
        5.2.5 Libreria>js>Vehiculos: En esta carpeta estaran todos los archivos js especiales para el modulo Vehiculos
    5.3 Libreria>cargar-header: Este archivo sirve para insertar el header en el codigo html cuando es llamado
    5.4 Libreria>modificar-header: Este archivo sirve para hacerle modificacione al header
    5.5 Libreria>PlantillaHTML: Este archivo sirve como guia para crear los formularios HTML

6. OrdenTrabajo: En esta carpetan estaran todos los formularios del modulo Orden de Trabajo
    -Se identifica con el prefijo: OT_
        Asesor: OT-A_
        Encargado de Bodega: OT-E_
        Jefe de Taller: OT-J_
        Tecnico: OT-T_

7. RRHH: En esta carpetan estaran todos los formularios del modulo Recursos Humanos
    -Se identifica con el prefijo: RH_
        Jefe de RRHH: RH-J_
        Asistente de RRHH: RH-A_

8. Vehiculos: En esta carpetan estaran todos los formularios del modulo Vehiculos
    -Se identifica con el prefijo: VE_
