Descripcion y estructura de Carpetas
1. DDLs: En esta carpeta estaran todos los scripts DDL necesarios para crear la Base de Datos
    
2. DMLs: En esta carpetan estaran todos los scripts DML necesarios para llenar la Base de Datos

3. Funciones: En esta carpetan estaran todas las funciones de utilidad que seran usadas por los procedimientios Almacenados
    - Se identifica con el prefijo: FN_

4. Modelo Relacional: En esta carpetan estaran todos los archivos que describen el Modelo Relacional de la Base de Datos

5. Procedimientos: En esta carpeta estaran todos los procedimientos almacenados respectivos para cada modulo
    5.1 Procedimientos>Facturacion: En esta carpeta estaran los procedimientos del modulo de Facturacion
    5.2 Procedimientos>Gestion de Usuario: En esta carpeta estaran los procedimientos del modulo de Gestion de Usuario
    5.3 Procedimientos>Orden de Trabajo: En esta carpeta estaran los procedimientos del modulo de Orden de Trabajo
        - Se identifica con el prefijo: OT_
            Asesor: OT-A_
            Encargado de Bodega: OT-E_
            Jefe de Taller: OT-J_
            Tecnico: OT-T_
    5.4 Procedimientos>RRHH: En esta carpeta estaran los procedimientos del modulo de En esta carpeta estaran los procedimientos del modulo de RRHH
    5.5 Procedimientos>Vehiculos: En esta carpeta estaran los procedimientos del modulo de En esta carpeta estaran los procedimientos del modulo de Vehiculos

6. Vistas: En esta carpeta estaran todas las vistaas que seran usadas por los procedimientos Almacenados
    - Se identifica con el prefijo: VI_
