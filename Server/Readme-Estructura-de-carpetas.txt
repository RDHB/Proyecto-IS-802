Descripcion y estructura de Carpetas
1. Conexion BD: En esta carpetan estaran los archivos necesarios para llevar a cabo la conexion con la Base de Datos

2. Conexion Fronted: En esta carpeta estaran todos los archivos necesarios para llevar a cabo la conexion con el Frontend

3. Configuracion Server: En esta carpetan estaran todos los archivos para la configuracion del servidor

4. Objetos: En esta carpeta estaran todos los archivos que contengan clases para la programacion orientada a objetos

6. Peticiones: En esta carpetan estaran todos archivos que atienden las peticiones AJAX

7. Procesos: En esta carpetan estaran todos los archivos los cuales requieran procesamiento de datos
