Descripcion y estructura de Carpetas
1. apis: En esta carpeta estaran todas las apis a ser llamadas
    - Se identifica con el prefijo: apis
    1.1 functionsMiscelaneos: En esta carpeta estaran todas las funciones para realizar alguna funcionalidad extra
        - Se identifica con el prefijo: functions
    1.2 functionsQuerys: En esta carpeta estaran todas funciones principales de todos los modulos
        - Se identifica con el prefijo: functions
    
2. db: En esta carpeta estaran las configuraciones para conectarse con la base de datos

3. others: En esta carpeta estaran las configuraciones para poder mostrar las pantallas del aplicativo

4. settings: En esta carpeta estaran las configuraciones del servidor

5. views: En esta carpeta estaran todas las rutas que hacen referencia a las respectivas pantallas del aplicativo
